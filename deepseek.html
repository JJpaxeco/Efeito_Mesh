Como Interagir com a Simulação

  • Movimente o cursor próximo ao tecido para simular o efeito do vento
  • Clique e arraste rapidamente sobre o tecido para rasgá-lo
  • Ajuste os controles deslizantes para modificar as propriedades físicas
  • Use os botões de vento para aplicar vento constante em uma direção
  • Pause a simulação para inspecionar o tecido em detalhes
  • Rasgue o tecido arrastando o mouse rapidamente através das molas
